HotellMenu är ett C#-projekt som hanterar menyer och funktioner för hotellverksamhet. Projektet är utvecklat med .NET 9 och använder objektorienterad programmering för att modellera menyer, rätter och relaterade funktioner.

Funktioner
•	Hantering av hotellmenyer och rätter
•	Möjlighet att lägga till, ta bort och visa menyer
•	Struktur för vidareutveckling med fler hotellrelaterade funktioner

Installation
Klona detta repository:
git clone https://github.com/sabinakarlsson/HotellMenu.git
Öppna projektet i Visual Studio 2022 eller senare.
Bygg och kör projektet.

Projektstruktur
•	Menu.cs – Klass för menyer
•	Dish.cs – Klass för rätter
•	Program.cs – Startpunkt för applikationen
•	HotellMenu.csproj – Projektfil för .NET 9

Användning
Starta programmet och följ instruktionerna i konsolen för att hantera hotellmenyer och rätter.
